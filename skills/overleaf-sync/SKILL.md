---
name: overleaf-sync
description: Sync a local directory with Overleaf. Use when you need to push or pull LaTeX project files between the local workspace and Overleaf.
---

## Steps

### 1. Install overleaf-for-agents

```bash
pip install overleaf-for-agents
# or
uv tool install overleaf-for-agents          # recommended with uv
```

### 2. Run leaf claude

```bash
leaf claude
```

If you run into issues, refer to https://github.com/yqi96/overleaf-for-agents.
